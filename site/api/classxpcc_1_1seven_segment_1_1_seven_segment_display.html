Detailed Description

template<typename Spi, typename Load, uint8_t DIGITS = 1>
class xpcc::sevenSegment::SevenSegmentDisplay< Spi, Load, DIGITS >

This driver makes seven segment displays that consists of several digits and are connected to daisy-chained shift registers (74xx595) easy to use.

It because the Load signal must be strobe when all data is shifted out this blocks until the SPI has finished.
