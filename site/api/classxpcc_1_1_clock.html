Detailed Description

This class is implemented using gettimeofday() from <sys/time.h> for any Unix-OS.

For Cortex-M targets the user has to enable the xpcc::SysTick timer.

For the AVRs targets the user has to use the increment() method to generate a suitable timebase, preferably by incrementing the time value inside a timer interrupt function.

Example:

// Interrupt every 1ms
ISR(TIMER)
{
xpcc::Clock::increment();
}
