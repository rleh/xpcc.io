Detailed Description

This implementation uses a ringbuffer. The size of the ringbuffer can be changed in the project.cfg file.

Just add add a value for tx_buffer and/or rx_buffer in the parameters section. The size can be any value between 1 and 254 for buffered and 0 for non-buffered operation.

Example:

[parameters]
uart.at90_tiny_mega.0.tx_buffer = 20
uart.at90_tiny_mega.0.rx_buffer = 30
