Detailed Description

template<typename T, template< typename > class Accessor = ::xpcc::accessor::Ram>
class xpcc::interpolation::Lagrange< T, Accessor >

Example:

typedef xpcc::Pair<float, float> Point;
// interpolate x^2 over the range of 1 <= x <= 3
Point points[3] =
{
{ 1, 1 },
{ 2, 4 },
{ 3, 9 }
};
xpcc::interpolation::Lagrange<Point> value(points, 3);
...
float output = value.interpolate(1.5f);
// output => 2.25;
See also
http://en.wikipedia.org/wiki/Lagrange_interpolation
Warning
Only floating points types are allowed as second type of xpcc::Pair, otherwise the calculation will deliver wrong results!
Template Parameters
TAny specialization of xpcc::Pair<> with a floating point type as second template argument.
AccessorAccessor class. Can be xpcc::accessor::Ram, xpcc::accessor::Flash or any self defined accessor class. Default is xpcc::accessor::Ram.
