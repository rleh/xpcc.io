Typedef Documentation

using xpcc::ShortPeriodicTimer = typedef GenericPeriodicTimer< ::xpcc::Clock, ShortTimestamp>

Extra care must be taken when not calling the isExpired() method for more than 32 seconds. Due to an overflow in the implementation this might add an additional delay of up to 32s ticks in the worst case. Always call restart() or restart(time) before reusing the timer to avoid this behaviour.

If you need a longer time period, use PeriodicTimer.
